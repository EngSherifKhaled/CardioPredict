# CardioPredict: AI-Powered Heart Disease Prevention  

Predict the presence of heart disease using patient health indicators such as
age, blood pressure, cholesterol, and chest pain type.

Dataset: Heart Disease UCI (Kaggle)

---

## Project Structure
- data/ → contains dataset files
- notebooks/ → Jupyter/Colab notebooks
- images/ → saved plots and figures

## Objective
Perform exploratory data analysis and later build a machine learning model to
classify patients as at risk (1) or not at risk (0) of heart disease.
